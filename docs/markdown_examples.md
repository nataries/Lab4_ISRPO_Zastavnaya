# Заголовки
Заголовки подразделяются на шесть уровней, создаются при помощи символа # (чем меньше количество символов, тем больше размер заголовка)
1. # Заголовки первого уровня
2. ## Заголовки второго уровня
3. ### Заголовки третьего уровня
4. #### Заголовки четвертого уровня
5. ##### Заголовки пятого уровня
6. ###### Заголовки шестого уровня

# Горизонтальные линии
---
Создаются при помощи трех подряд идущих символов * (или -) для разделения блоков текста.

# Форматирование
---
Markdown поддерживает несколько видов форматирования текста:
1. **жирный текст** создаются с помощью обрамления текста символами ** с обеих сторон.
2. *для курсива* необходимо заключить слово в звездочки \*   \*
3. ~~зачеркнутый~~ текст создается при помощи двойного символа ~.
4. `моноширный текст создается при помощи бэктиков`

# Списки
Списки подразделяются на нумерованные, маркированные и вложенные.
1. В нумерованных списках каждый новый пункт является цифрой
2. Применяются, когда последовательность имеет значение 
---
+ В маркированных списках каждый новый пункт является точкой
+ Полезно, когда нужно оформить информацию в виде списка, но порядок не имеет значения
---
1. Во вложенных списках имеются подпункты
    + Применяются, когда основной пункт подразделяется на подпункты
2. А дальше можно вновь писать основные пункты
