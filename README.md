## Структура проекта
Структура проекта включает 5 папок и файл README.md с описанием проекта. 
1. Папка `client` содержит файлы с кодом для фронтенда: `about.html` и `index.html`
2. Папка `docs` содержит файлы `latex_examples.md` `markdown_examples.md`, в которых приведены примеры элементов с использованием Markdown
3. Папка `repo` содержит скриншоты, демонстрирующие вид проекта в ходе его выполнения
4. Папка `server` содержит программный файл бэкенда с реализованной программой на языке C#
5. Папка `terminal_practice` содержит две папки:
    + Папка `data` содержит текстовый файл `text.txt`
    + Папка `logs` содержит файл `app.log`

Также в ней есть два текстовых файла: `data.txt` и `logs.txt`
6. Файл `README.md` с описанием проекта

+ **Код**
Реализация конструкции switch-case в коде консольного приложения этого проекта:
```
switch (choice)
    {
        case "1":
            Console.WriteLine($"ФИО - {myName}");
            break;

        case "2":
            Console.WriteLine($"Группа - {group}");
            break;
        case "3":
            Console.WriteLine($"Текущая дата: {curDate}");
            break;
        case "4":
            Console.WriteLine("Вы вышли из программы!");
            return;
        default:
            Console.WriteLine("Ошибка, такого варианта нет");
            break;
    }
```
